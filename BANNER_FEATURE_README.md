# Header Drawer 自定义 Banner 功能使用说明

## 功能概述

在 header-drawer（抽屉菜单）的子菜单中添加了自定义 Banner 图片功能，可以在菜单中展示自定义的横幅广告。

## 如何使用

### 1. 在 Shopify 后台配置

1. 进入 **在线商店 > 主题 > 自定义**
2. 选择 **Header（页眉）** 部分
3. 找到菜单块（Menu Block）设置
4. 在 **"媒体类型"（Media Type）** 下拉选项中，选择 **"自定义Banner图"**

### 2. Banner 设置选项

选择 "自定义Banner图" 后，会出现以下新的设置选项：

#### Banner图片
- 点击上传您的 Banner 图片
- **推荐尺寸**: 1200x400px
- 支持常见图片格式（JPG, PNG, WebP等）

#### Banner链接
- 可选项：设置点击 Banner 后跳转的链接
- 可以是商品页面、系列页面或任何自定义URL
- 留空则 Banner 不可点击

#### Banner图片比例
选择 Banner 的显示比例：
- **自动（Auto）**: 保持图片原始比例
- **16:9**: 宽屏比例（默认推荐）
- **21:9**: 超宽屏比例
- **3:1**: 横向长条比例

#### Banner布局方式（桌面端）
选择 Banner 与菜单的排列方式：
- **上下排列（默认）**: Banner 显示在菜单下方
- **左右排列**: Banner 与菜单并排显示（仅桌面端生效）
- **注意**: 移动端始终为上下排列以保证最佳体验

### 3. Banner 显示位置

Banner 会显示在以下位置：
- **三级菜单**: 在子菜单项下方显示
- **顶级菜单**: 在抽屉底部显示（仅桌面端）

## 技术实现

### 修改的文件

1. **blocks/_header-menu.liquid**
   - 添加了 `custom_banner` 选项到 `menu_style` 选择器
   - 添加了 Banner 图片上传、链接和比例设置
   - 添加了相关 CSS 样式

2. **snippets/header-drawer.liquid**
   - 在子菜单区域添加了 Banner 渲染逻辑
   - 在底部推荐区域添加了 Banner 支持
   - 添加了 Banner 的样式定义

### 样式特性

- **响应式**: Banner 会自动适应不同屏幕尺寸
- **圆角**: 继承菜单图片圆角设置
- **悬停效果**: 可点击的 Banner 悬停时会有轻微放大效果
- **懒加载**: Banner 图片采用懒加载，优化页面性能
- **灵活布局**: 
  - 桌面端支持上下排列和左右排列两种布局方式
  - 移动端自动切换为上下排列
  - 左右排列时，Banner 与菜单平分空间，保持良好的视觉平衡

## 使用场景

- 促销活动宣传
- 新品推荐
- 品牌故事展示
- 特殊优惠信息
- 季节性营销活动

## 注意事项

1. **图片优化**: 上传前请优化图片大小，建议不超过 200KB
2. **移动端**: 在移动端默认不显示底部 Banner（eager_loading=false）
3. **Alt 文本**: 建议在 Shopify 图片设置中添加 Alt 文本，提升 SEO 和可访问性
4. **兼容性**: 该功能兼容现有的 collection_images、featured_products 等其他菜单样式

## 示例配置

### 示例1：促销横幅（上下排列）
```
媒体类型: 自定义Banner图
Banner图片: [上传 1200x400px 的促销图片]
Banner链接: /collections/sale
Banner图片比例: 16:9
Banner布局方式: 上下排列
图片圆角: 8px
```

### 示例2：品牌展示（左右排列）
```
媒体类型: 自定义Banner图
Banner图片: [上传 800x600px 的品牌图片]
Banner链接: /pages/about
Banner图片比例: 16:9
Banner布局方式: 左右排列
图片圆角: 12px
```
